# ⚠️ КРИТИЧЕСКИ ВАЖНО: Проверка точного Referer

## Проблема
Все домены добавлены, но фото все еще не загружаются (403).

## Решение: Проверить ТОЧНЫЙ Referer

### Шаг 1: Найдите точный Referer

1. Откройте консоль браузера (F12) → **Network**
2. Найдите запрос к `photo?maxwidth=800&photoreference=...` с ошибкой 403
3. Кликните на него
4. Откройте вкладку **Headers**
5. В разделе **Request Headers** найдите **Referer**
6. **Скопируйте ТОЧНОЕ значение** - это критически важно!

### Шаг 2: Проверьте формат домена

**Если Referer показывает:**
- `https://www.flip-trip.com/` (со слешем в конце)

**То в Application Restrictions должно быть:**
- `www.flip-trip.com/*` (без https://, с /*)

### Шаг 3: Попробуйте добавить ВСЕ варианты

Иногда Google требует явного указания, даже если wildcard должен покрывать:

1. `flip-trip.com/*` (без www)
2. `www.flip-trip.com/*` (с www)
3. `*.flip-trip.com/*` (wildcard для всех поддоменов)
4. `flip-trip.com` (БЕЗ /* - попробуйте и этот вариант)
5. `www.flip-trip.com` (БЕЗ /* - попробуйте и этот вариант)

### Шаг 4: Проверьте, нет ли других доменов

Если вы используете:
- Preview deployments на Vercel
- Staging environment
- Другие поддомены

Добавьте их тоже!

### Шаг 5: Временный тест

Для проверки, что проблема точно в доменах:

1. В **Application restrictions** выберите **"None"** (временно!)
2. Сохраните
3. Обновите страницу
4. Если фото загружаются - значит проблема точно в доменах
5. Вернитесь к **"Websites"** и добавьте домены заново

### Шаг 6: Проверьте API Restrictions

Убедитесь, что в **API Restrictions** включены:
- ✅ **Places API (New)** ← это критически важно для фото!
- ✅ Places API (старый)
- ✅ Maps JavaScript API
- ✅ Geocoding API

### Шаг 7: Подождите дольше

Изменения в Google Cloud Console могут применяться:
- Обычно: 1-2 минуты
- Иногда: до 10-15 минут
- После изменений: подождите минимум 5 минут перед проверкой

## Альтернативное решение

Если ничего не помогает, можно создать прокси на бэкенде для загрузки фото через серверный ключ (без ограничений по доменам).
