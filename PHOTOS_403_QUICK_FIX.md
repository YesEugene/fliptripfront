# ⚡ Быстрое исправление 403 для фотографий

## Проблема
Добавили `https://www.flip-trip.com/`, но все равно 403 ошибка.

## Решение

### ❌ Неправильный формат:
```
https://www.flip-trip.com/
```

### ✅ Правильный формат:
```
www.flip-trip.com/*
```

## Что нужно сделать:

1. **Удалите** неправильную запись `https://www.flip-trip.com/` (если есть)

2. **Добавьте ОБЕ записи:**
   - `flip-trip.com/*` (без www)
   - `www.flip-trip.com/*` (с www)

3. **Важно:**
   - ❌ БЕЗ `https://`
   - ✅ С `/*` в конце
   - ✅ Оба варианта (с www и без)

4. **Также добавьте:**
   - `*.fliptripfrontend.vercel.app/*` (для Vercel deployments)
   - `localhost:5173/*` (для разработки)

5. Нажмите **"SAVE"**

6. Подождите 1-2 минуты

7. Обновите страницу (Ctrl+F5 или Cmd+Shift+R для полной перезагрузки)

## Проверка

После сохранения:
1. Откройте консоль браузера (F12) → Network
2. Найдите запрос к `/place/photo`
3. Проверьте Response Status - должен быть **200**, а не 403
4. Если все еще 403:
   - Проверьте, что формат правильный (без `https://`, с `/*`)
   - Убедитесь, что оба домена добавлены (с www и без)
   - Подождите еще 1-2 минуты (изменения могут применяться с задержкой)
